
# BeluEssence Creatives

This template should help get you started developing with Vue 3 in Vite.

## Run Locally 


### Clone the project

bash
```sh
  git clone https://gitlab.com/sunum/rslab-frontend-admin.git](https://github.com/leandrodzn/beluessence-creatives-front.git
```

### Go to the project directory

bash
```sh
  cd beluessence-creatives
```

### Create your branch
bash
```sh
  git branch name/develop // example: leandro/develop, wilder/develop
```

### Change to your branch
bash
```sh
  git checkout name/develop
```

### Install dependencies

bash
```sh
  npm install
```

### Start the server

bash
```sh
  npm run dev
```

### Push your branch
bash
```sh
  git push origin name/develop
```

#### Don't forget to set your global profile in Git
